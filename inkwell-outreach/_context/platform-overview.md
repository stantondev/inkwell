# Inkwell Platform Overview

## What Inkwell Is (Plain Language)
<!-- FILL THIS IN: 2-3 sentences explaining Inkwell to someone who's never heard of it -->
Inkwell is a federated social journaling platform. Think LiveJournal meets MySpace, rebuilt for 2026 with modern web tech and ActivityPub federation. No algorithms, no ads, user-owned spaces.

## What's Working Well Right Now
<!-- FILL IN: Be honest about what you're proud of -->
- Rich text editor with 17+ formatting options, cover images, excerpts, word count
- Full profile customization (8 themes, custom colors, fonts, layouts, music, avatar frames, custom HTML/CSS)
- ActivityPub federation (followable from Mastodon and other fediverse instances)
- Journal-style page-turning feed UI
- Stamps (like postage stamps, not generic likes) and Inks (community discovery)
- Writing Circles (group discussion spaces with a Writer's Salon aesthetic)
- Newsletter delivery built in (no Substack needed)
- Guestbooks, top pen pals, status messages
- Community roadmap where users vote on features
- Data import from WordPress, Medium, Substack, and more
- Postage (reader support payments via Stripe Connect)
- Series and collections for organizing entries
- Cross-posting to Mastodon from the editor
- Custom domains are available for Plus subscribers

## What's Honestly Still Rough
<!-- FILL IN: Be transparent about gaps -->
- Small community (early access stage)
- Search uses basic Postgres (Meilisearch not deployed yet)
- No mobile app (responsive web only)
- Fediverse relay content can be hit-or-miss quality

## Features Writers Specifically Care About
- Rich editor with distraction-free writing mode (ruled-line paper aesthetic)
- Entry privacy controls (public, friends-only, private, custom filters)
- Full profile ownership (themes, custom HTML/CSS like MySpace)
- No algorithm deciding who sees your writing
- Built-in newsletter (don't need a separate Substack)
- Version history for entries
- Data export and import (you can always leave)
- Connected to the fediverse (your writing reaches Mastodon users)

## Current Stage
<!-- FILL IN: Be specific -->
- Early access / public beta
- Current user count: 40
- Solo developer building in public
